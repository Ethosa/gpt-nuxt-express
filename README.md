## Демо:
https://ethosa.github.io/gpt-nuxt-express  (Бекенд в любом случае придется развернуть локально)

## Запуск

### Backend
> Не забудьте создать `.env` файл!
Для запуска бэкенда:
```
cd backend
pnpm i && pnpm dev
```


### Frontend
Для запуска фронтенда:
```
cd frontend
pnpm i && pnpm dev
```
